# Utilidades

## Instalación
```sh
composer install sebacarrasco93/utilidades
```

## Uso
```php
<?php
return Utilidades::{nombre_método}({parámetro(s)});
```

## Métodos disponibles:

### Integers
#### Peso chileno

Agrega el símbolo $, y agrega puntos

```php
<?php
return Utilidades::peso_chileno(1500); // devuelve $1.500
```
